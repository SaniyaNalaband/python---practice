# 💻 1. What is VS Code?

**Visual Studio Code (VS Code)** is a free, lightweight, and powerful source code editor developed by **Microsoft**.

It is one of the most popular code editors in the world because it supports many programming languages through extensions.

You can write code in:

- 🐍 Python
- 🌐 HTML
- 🎨 CSS
- ⚡ JavaScript
- ☕ Java
- ⚙️ C
- ⚙️ C++
- 🦀 Rust
- 🐹 Go
- and many more.

---

# 🤔 2. Why Do We Need VS Code?

Although Python comes with **IDLE**, professional developers prefer VS Code because it provides many useful features.

Some of its advantages are:

- ✅ Intelligent code suggestions (IntelliSense)
- ✅ Syntax highlighting
- ✅ Error detection
- ✅ Built-in terminal
- ✅ Git & GitHub integration
- ✅ Extension support
- ✅ Debugging tools
- ✅ Fast and lightweight
- ✅ Cross-platform

# 🧩 6. Installing the Python Extension

VS Code does not understand Python automatically.

Install the official Python extension.

Steps:

1. Open VS Code.
2. Click the **Extensions** icon.
3. Search for **Python**.
4. Install the extension published by **Microsoft**.

This extension provides:

- Syntax highlighting
- Auto-completion
- Debugging
- Code formatting
- IntelliSense

---

# 🐍 7. Selecting the Python Interpreter

VS Code needs to know which Python installation to use.

Steps:

1. Press **Ctrl + Shift + P**.
2. Type:

```
Python: Select Interpreter
```

3. Choose the installed Python version.

Example:

```
Python 3.12 (64-bit)
```

# ▶️ 8. Running Your First Program

Create a file:

```text
hello.py
```

Write:

```python
print("Hello, VS Code!")
```

Run the program by:

- Clicking the ▶ Run button
- Or using the terminal:

```bash
python hello.py
```

Output:

```text
Hello, VS Code!
```

# ⚠️ 11. Common Problems

### Problem:

```
Python interpreter not found
```

**Solution:**

Select the correct interpreter using:

```
Python: Select Interpreter
```

---

### Problem:

```
python is not recognized
```

**Solution:**

Ensure Python is installed and added to PATH.

---

### Problem:

Code does not run.

**Solution:**

Check:

- Python is installed.
- Python extension is installed.
- Correct interpreter is selected.
- File is saved with `.py` extension.

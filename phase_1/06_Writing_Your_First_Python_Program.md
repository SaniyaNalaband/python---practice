# 📖 1. What is a Python Program?

A **Python program** is a collection of Python instructions written in a file with the extension:

```text
.py
```

Examples:

```text
hello.py
calculator.py
student.py
game.py
```

A Python program tells the computer **what to do**.

---

# 🤔 2. Your First Python Program

Create a new file called:

```text
hello.py
```

Write the following code:

```python
print("Hello, World!")
```

Save the file.

Run it.

Output:

```text
Hello, World!
```

Congratulations! 🎉

You have written your very first Python program.

---

# 🧠 3. Understanding the Code

```python
print("Hello, World!")
```

Let's understand every part.

### `print`

`print` is a **built-in Python function**.

Its purpose is to display information on the screen.

---

### `( )`

Parentheses hold the values passed to the function.

Everything inside the parentheses is printed.

---

### `"Hello, World!"`

This is called a **string**.

A string is simply text enclosed inside quotation marks.

---

# 📌 4. Syntax

General syntax:

```python
print(value)
```

Examples:

```python
print("Python")
```

```python
print(100)
```

```python
print(3.14)
```

```python
print(True)
```

# 📝 7. Strings

A **string** is any text enclosed inside quotation marks.

Examples:

```python
"Python"
```

```python
"Hello"
```

```python
"My Name is Saniya"
```

You can also use single quotes.

```python
'Python'
```

Both are valid.

---

# ⚠️ 8. Common Beginner Mistakes

## Mistake 1

```python
print(Hello)
```

Error:

```text
NameError
```

Correct:

```python
print("Hello")
```

---

## Mistake 2

```python
Print("Hello")
```

Error:

```text
NameError
```

Python is **case-sensitive**.

Correct:

```python
print("Hello")
```

---

## Mistake 3

```python
print("Hello"
```

Error:

```text
SyntaxError
```

Correct:

```python
print("Hello")
```

---

## Mistake 4

Saving the file as:

```
hello.txt
```

instead of

```
hello.py
```

---

# 💡 9. Best Practices

✅ Use meaningful file names.

Good:

```text
calculator.py
```

Bad:

```text
abc.py
```

---

Always save the file before running it.

---

Use lowercase file names.

---

# 📌 10. Key Notes

- Every Python file ends with `.py`.
- `print()` displays output.
- Strings must be inside quotes.
- Python is case-sensitive.
- Save your file before running it.

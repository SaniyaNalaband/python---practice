# 📖 1. What is Python?

Python is a **high-level**, **interpreted**, and **general-purpose programming language** designed to be simple, readable, and easy to learn.

Python allows programmers to build many different kinds of software, including:

- 🌐 Websites
- 🖥️ Desktop Applications
- 🤖 Artificial Intelligence (AI)
- 📊 Data Analysis Programs
- 🧠 Machine Learning Models
- 🎮 Games
- ⚙️ Automation Scripts
- 🔒 Cybersecurity Tools
- 🌍 APIs
- 📡 Internet of Things (IoT) Applications

Think of Python as a language that helps humans communicate instructions to computers.

# 🐍 3. Why is it Called Python?

Many beginners think Python is named after the snake.

❌ That is **not true**.

The name comes from the British comedy television show:

> **Monty Python's Flying Circus**

Guido van Rossum liked the name because it was:

- Short
- Unique
- Easy to remember

---

# 💻 4. What is a Programming Language?

A programming language is a language used to write instructions that a computer can understand and execute.

## Example

```python
print("Hello")
```

### Output

```text
Hello
```

Without programming languages, computers understand only **machine language (0s and 1s).**

Python translates your instructions into something the computer can execute.

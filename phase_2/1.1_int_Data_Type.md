# 📖 1. What is an Integer?

An **Integer (`int`)** is a whole number **without a decimal point**.

Integers can be:

- Positive
- Negative
- Zero

Examples:

```python
10
250
9999
```

```python
0
```

```python
-5
-100
-999
```

# 🧠 7. Integer Immutability

Integers are **immutable**.

That means their value cannot be changed after creation.

Example:

```python
x = 10

print(id(x))

x = x + 5

print(id(x))
```

Possible Output:

```
140620240992488
140620240992648
```

The IDs are different because Python creates a **new integer object**.

# 🔄 8. Integer Type Conversion

Convert a float to an integer:

```python
print(int(5.9))
```

Output:

```
5
```

The decimal part is removed (truncated), **not rounded**.

---

Convert a string:

```python
print(int("100"))
```

Output:

```
100
```

# ⚠️ 9. Common Mistakes

## ❌ Writing Commas

Wrong:

```python
num = 10,000
```

This creates a **tuple**, not an integer.

Correct:

```python
num = 10000
```

Or use underscores:

```python
num = 10_000
```

---

## ❌ Using Quotes

```python
age = "20"
```

This is a **string**, not an integer.

Correct:

```python
age = 20
```

---

## ❌ Expecting Division to Return an Integer

```python
print(10 / 2)
```

Output:

```
5.0
```

If you want an integer result (when appropriate), use floor division:

```python
print(10 // 2)
```

Output:

```
5
```

# 💡 10. Best Practices

✅ Use meaningful variable names.

```python
student_count = 50
```

Instead of:

```python
x = 50
```

---

✅ Use underscores for readability.

```python
population = 1_450_000_000
```

This is exactly the same as:

```python
population = 1450000000
```

---

# 🚀 11. Pro Tips

Python automatically removes underscores.

```python
number = 1_000_000

print(number)
```

Output:

```
1000000
```

Underscores make large numbers easier to read.

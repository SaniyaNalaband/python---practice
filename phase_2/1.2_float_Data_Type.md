# 📖 1. What is a Float?

A **Float (`float`)** is a number that contains a **decimal point**.

Examples:

```python
10.5
99.99
0.5
100.0
-45.75
```

Unlike integers, floats can represent fractional values.

# 📊 4. Float vs Integer

```python
a = 10
b = 10.0

print(type(a))
print(type(b))
```

Output:

```text
<class 'int'>
<class 'float'>
```

Even though both represent ten, `10` is an integer and `10.0` is a float.

# 🔄 6. Type Conversion

Convert an integer to a float:

```python
print(float(10))
```

Output:

```text
10.0
```

---

Convert a string:

```python
print(float("99.5"))
```

Output:

```text
99.5
```

---

Convert a float to an integer:

```python
print(int(8.9))
```

Output:

```text
8
```

The decimal part is removed, not rounded.

---

# ⚠️ 7. Floating-Point Precision

Computers store floating-point numbers in binary.

Because of this, some decimal numbers cannot be represented exactly.

Example:

```python
print(0.1 + 0.2)
```

Output:

```text
0.30000000000000004
```

This surprises many beginners, but it's normal.

---

# 🔬 8. Scientific Notation

Python supports scientific notation using `e` or `E`.

```python
a = 3e2
print(a)
```

Output:

```text
300.0
```

---

```python
b = 5.2e3
print(b)
```

Output:

```text
5200.0
```

---

```python
c = 7e-2
print(c)
```

Output:

```text
0.07
```

# 💡 12. Best Practices

✅ Use floats for measurements, prices, percentages, and scientific values.

---

# 🚀 13. Pro Tips

Instead of:

```python
a = 0.1 + 0.2

print(a == 0.3)
```

Use a tolerance:

```python
a = 0.1 + 0.2

print(abs(a - 0.3) < 1e-9)
```

Output:

```text
True
```

This is a common technique in scientific and engineering programs.

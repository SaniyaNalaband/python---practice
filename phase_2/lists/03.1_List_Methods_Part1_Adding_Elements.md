# 📖 1. `append()`

Adds **one element** to the **end** of the list.

### Syntax

```python
list.append(item)
```

---

## Example 1

```python
fruits = ["Apple", "Banana"]

fruits.append("Mango")

print(fruits)
```

Output

```text
['Apple', 'Banana', 'Mango']
```

---

## Example 2

```python
numbers = [10, 20]

numbers.append(30)

print(numbers)
```

Output

```text
[10, 20, 30]
```

---

## Append Different Data Types

```python
data = [1, 2]

data.append(True)

print(data)
```

Output

```text
[1, 2, True]
```

---

## Append Another List

```python
a = [1, 2]
b = [3, 4]

a.append(b)

print(a)
```

Output

```python
[1, 2, [3, 4]]
```

Notice that `b` is added as **one nested element**.

---

# 📖 2. `extend()`

Adds **all elements** from another iterable.

### Syntax

```python
list.extend(iterable)
```

---

## Example 1

```python
a = [1, 2]
b = [3, 4]

a.extend(b)

print(a)
```

Output

```text
[1, 2, 3, 4]
```

Unlike `append()`, `extend()` adds each element individually.

---

## Extend with a Tuple

```python
numbers = [1, 2]

numbers.extend((3, 4))

print(numbers)
```

Output

```text
[1, 2, 3, 4]
```

---

## Extend with a String

```python
letters = ["A"]

letters.extend("BC")

print(letters)
```

Output

```text
['A', 'B', 'C']
```

Each character is treated as an individual element.

---

# 📖 3. `insert()`

Inserts an element at a specific index.

### Syntax

```python
list.insert(index, item)
```

---

## Example 1

```python
fruits = ["Apple", "Mango"]

fruits.insert(1, "Banana")

print(fruits)
```

Output

```text
['Apple', 'Banana', 'Mango']
```

---

## Insert at the Beginning

```python
numbers = [20, 30]

numbers.insert(0, 10)

print(numbers)
```

Output

```text
[10, 20, 30]
```

---

## Insert Beyond the Last Index

```python
numbers = [1, 2]

numbers.insert(100, 3)

print(numbers)
```

Output

```text
[1, 2, 3]
```

Python inserts it at the end.

---

## Insert Using a Negative Index

```python
numbers = [10, 20, 40]

numbers.insert(-1, 30)

print(numbers)
```

Output

```text
[10, 20, 30, 40]
```

The new element is inserted **before** the last element.

# 📊 append() vs extend() vs insert()

| Feature | `append()` | `extend()` | `insert()` |
|---------|------------|------------|------------|
| Adds one item | ✅ | ❌ | ✅ |
| Adds multiple items | ❌ | ✅ | ❌ |
| Position | End | End | Any index |
| Accepts iterable | As one item | Expands iterable | As one item |

---

### Example Comparison

```python
a = [1, 2]
b = [3, 4]

a.append(b)
print(a)
```

Output

```python
[1, 2, [3, 4]]
```

---

```python
a = [1, 2]
b = [3, 4]

a.extend(b)
print(a)
```

Output

```python
[1, 2, 3, 4]
```

# ⚠️ Common Mistakes

## ❌ Assigning the Result

```python
numbers = [1, 2]

result = numbers.append(3)

print(result)
```

Output

```text
None
```

Why?

`append()` modifies the original list and returns `None`.

Correct:

```python
numbers.append(3)

print(numbers)
```

---

## ❌ Confusing `append()` and `extend()`

```python
a = [1, 2]

a.append([3, 4])

print(a)
```

Output

```python
[1, 2, [3, 4]]
```

---

```python
a = [1, 2]

a.extend([3, 4])

print(a)
```

Output

```python
[1, 2, 3, 4]
```

---

# 💡 Best Practices

- Use `append()` to add a single item.
- Use `extend()` to combine collections.
- Use `insert()` only when position matters.

---

# 🚀 Pro Tips

You can build a list using `append()` inside a loop.

```python
numbers = []

for i in range(1, 6):
    numbers.append(i)

print(numbers)
```

Output

```text
[1, 2, 3, 4, 5]
```

---

# 🧠 Memory Trick

```

↓

↓

End

-----------------

extend()

↓

Many items

↓

End

-----------------

insert()

↓

One item

↓

Anywhere
```

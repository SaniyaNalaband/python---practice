# 📖 Why Do We Need Searching Methods?

Suppose you have a list of students:

```python
students = ["Saniya", "Rahul", "Aisha", "Rahul"]
```

You may want to know:

- Where is `"Aisha"`?
- How many times does `"Rahul"` appear?

Python provides `index()` and `count()` for these tasks.

---

# 📖 1. `index()`

Returns the **index (position)** of the **first occurrence** of a value.

### Syntax

```python
list.index(value)
```

---

## Example 1

```python
fruits = ["Apple", "Banana", "Mango"]

print(fruits.index("Banana"))
```

Output

```text
1
```

---

## Example 2 (Duplicates)

```python
numbers = [10, 20, 30, 20, 40]

print(numbers.index(20))
```

Output

```text
1
```

Only the **first occurrence** is returned.

---

## Example 3 – Search from a Specific Position

Syntax:

```python
list.index(value, start, end)
```

```python
numbers = [10, 20, 30, 20, 40]

print(numbers.index(20, 2))
```

Output

```text
3
```

Python starts searching from index `2`.

---

## Value Not Found

```python
fruits = ["Apple", "Banana"]

print(fruits.index("Orange"))
```

Output

```text
ValueError: 'Orange' is not in list
```

---

# 📖 2. `count()`

Returns the **number of times** a value appears.

### Syntax

```python
list.count(value)
```

---

## Example 1

```python
numbers = [10, 20, 20, 30, 20]

print(numbers.count(20))
```

Output

```text
3
```

---

## Example 2

```python
fruits = ["Apple", "Banana", "Apple"]

print(fruits.count("Apple"))
```

Output

```text
2
```

---

## Value Not Found

```python
numbers = [10, 20, 30]

print(numbers.count(50))
```

Output

```text
0
```

Unlike `index()`, `count()` **does not raise an error** if the value is not found.

# 📊 `index()` vs `count()`

| Feature | `index()` | `count()` |
|---------|-----------|-----------|
| Purpose | Find position | Count occurrences |
| Returns | Index (int) | Count (int) |
| Duplicates | First occurrence only | Counts all |
| Not Found | `ValueError` | `0` |

---

### Example Comparison

```python
numbers = [10, 20, 20, 30]
```

```python
print(numbers.index(20))
```

Output

```text
1
```

---

```python
print(numbers.count(20))
```

Output

```text
2
```

---

# ⚠️ Common Mistakes

## ❌ Assuming `index()` Returns All Positions

```python
numbers = [10, 20, 20]

print(numbers.index(20))
```

Output

```text
1
```

It returns **only the first occurrence**.

---

## ❌ Not Handling Missing Values

```python
fruits = ["Apple", "Banana"]

print(fruits.index("Orange"))
```

Raises:

```text
ValueError
```

Safe approach:

```python
if "Orange" in fruits:
    print(fruits.index("Orange"))
else:
    print("Not Found")
```

---

# 💡 Best Practices

- Use `index()` when you need the position.
- Use `count()` when you need the frequency.
- Check with `in` before calling `index()` if the value may not exist.

---

# 🚀 Pro Tips

Find all indexes of a value:

```python
numbers = [10, 20, 30, 20, 40, 20]

indexes = []

for i, value in enumerate(numbers):
    if value == 20:
        indexes.append(i)

print(indexes)
```

Output

```text
[1, 3, 5]
```

# 📖 Why Do We Need These Methods?

Suppose you have a shopping cart:

```python
cart = ["Milk", "Bread", "Eggs", "Butter"]
```

After purchasing the items, you may want to:

- Remove "Eggs"
- Remove the last item
- Empty the cart
- Delete the cart completely

Python provides different ways to do each of these tasks.

---

# 📖 1. `remove()`

Removes the **first occurrence** of a specified value.

### Syntax

```python
list.remove(value)
```

---

## Example 1

```python
fruits = ["Apple", "Banana", "Mango"]

fruits.remove("Banana")

print(fruits)
```

Output

```text
['Apple', 'Mango']
```

---

## Example 2 (Duplicates)

```python
numbers = [10, 20, 20, 30]

numbers.remove(20)

print(numbers)
```

Output

```text
[10, 20, 30]
```

Only the **first** `20` is removed.

---

## Value Not Found

```python
numbers = [10, 20, 30]

numbers.remove(50)
```

Output

```text
ValueError: list.remove(x): x not in list
```

---

# 📖 2. `pop()`

Removes and **returns** an element.

### Syntax

```python
list.pop(index)
```

If no index is given, it removes the **last element**.

---

## Example 1

```python
fruits = ["Apple", "Banana", "Mango"]

item = fruits.pop()

print(item)
print(fruits)
```

Output

```text
Mango
['Apple', 'Banana']
```

---

## Example 2

```python
numbers = [10, 20, 30]

removed = numbers.pop(1)

print(removed)
print(numbers)
```

Output

```text
20
[10, 30]
```

---

## Invalid Index

```python
numbers = [10, 20]

numbers.pop(5)
```

Output

```text
IndexError: pop index out of range
```

---

# 📖 3. `clear()`

Removes **all elements** from the list.

### Syntax

```python
list.clear()
```

---

## Example

```python
numbers = [10, 20, 30]

numbers.clear()

print(numbers)
```

Output

```text
[]
```

The list still exists—it is just empty.

---

# 📖 4. `del` Keyword

> **Note:** `del` is a **Python keyword**, not a list method.

It can:

- Delete an element
- Delete a slice
- Delete the entire list

---

## Delete One Element

```python
numbers = [10, 20, 30, 40]

del numbers[1]

print(numbers)
```

Output

```text
[10, 30, 40]
```

---

## Delete Multiple Elements

```python
numbers = [10, 20, 30, 40, 50]

del numbers[1:4]

print(numbers)
```

Output

```text
[10, 50]
```

---

## Delete Entire List

```python
numbers = [10, 20, 30]

del numbers

print(numbers)
```

Output

```text
NameError: name 'numbers' is not defined
```

The variable no longer exists.

# 📊 Comparison Table

| Method | Removes By | Returns Removed Item? | Error If Missing? |
|---------|------------|----------------------|-------------------|
| `remove()` | Value | ❌ No | ✅ `ValueError` |
| `pop()` | Index | ✅ Yes | ✅ `IndexError` |
| `clear()` | All elements | ❌ No | ❌ No |
| `del` | Index / Slice / List | ❌ No | ✅ If invalid |

# ⚠️ Common Mistakes

## ❌ Using `remove()` with an Index

Wrong:

```python
numbers = [10, 20, 30]

numbers.remove(1)
```

Python searches for the **value** `1`, not index `1`.

Correct:

```python
numbers.pop(1)
```

---

## ❌ Expecting `clear()` to Delete the List

```python
numbers = [1, 2]

numbers.clear()

print(numbers)
```

Output

```text
[]
```

The list still exists.

---

## ❌ Using `pop()` Without Saving the Result

```python
numbers = [10, 20, 30]

numbers.pop()

print(numbers)
```

Output

```text
[10, 20]
```

If you need the removed value:

```python
removed = numbers.pop()
```

---

# 💡 Best Practices

- Use `remove()` when you know the value.
- Use `pop()` when you know the index or need the removed element.
- Use `clear()` to empty a list.
- Use `del` to remove slices or delete the entire list.

---

# 🚀 Pro Tips

Safely remove a value:

```python
numbers = [10, 20, 30]

if 20 in numbers:
    numbers.remove(20)

print(numbers)
```

Output

```text
[10, 30]
```

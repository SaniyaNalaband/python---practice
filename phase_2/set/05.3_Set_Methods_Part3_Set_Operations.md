# 📖 What Are Set Operations?

Suppose two students know different programming languages.

```python
student1 = {"Python", "Java", "SQL"}
student2 = {"Python", "C++", "JavaScript"}
```

You may want to know:

- All languages known.
- Languages both know.
- Languages only student1 knows.
- Languages known by exactly one student.

Python provides four powerful methods.

---

# 📖 1. `union()`

Returns a **new set** containing all unique elements from both sets.

### Syntax

```python
set1.union(set2)
```

or

```python
set1 | set2
```

---

## Example

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a.union(b))
```

Output

```text
{1, 2, 3, 4, 5}
```

---

### Using the `|` Operator

```python
print(a | b)
```

Output

```text
{1, 2, 3, 4, 5}
```

---

# 📖 2. `intersection()`

Returns elements common to both sets.

### Syntax

```python
set1.intersection(set2)
```

or

```python
set1 & set2
```

---

## Example

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a.intersection(b))
```

Output

```text
{3}
```

---

### Using `&`

```python
print(a & b)
```

Output

```text
{3}
```

---

# 📖 3. `difference()`

Returns elements present in the first set but not in the second.

### Syntax

```python
set1.difference(set2)
```

or

```python
set1 - set2
```

---

## Example

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a.difference(b))
```

Output

```text
{1, 2}
```

---

### Reverse Difference

```python
print(b.difference(a))
```

Output

```text
{4, 5}
```

Difference depends on the order of the sets.

---

### Using `-`

```python
print(a - b)
```

Output

```text
{1, 2}
```

---

# 📖 4. `symmetric_difference()`

Returns elements that are in **either set**, but **not in both**.

### Syntax

```python
set1.symmetric_difference(set2)
```

or

```python
set1 ^ set2
```

---

## Example

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a.symmetric_difference(b))
```

Output

```text
{1, 2, 4, 5}
```

---

### Using `^`

```python
print(a ^ b)
```

Output

```text
{1, 2, 4, 5}
```

# 📊 Comparison Table

| Method | Operator | Meaning |
|---------|----------|---------|
| `union()` | `|` | All unique elements |
| `intersection()` | `&` | Common elements |
| `difference()` | `-` | Only in first set |
| `symmetric_difference()` | `^` | In either set, not both |

# ⚠️ Common Mistakes

## ❌ Thinking `difference()` is Symmetric

```python
a = {1, 2}
b = {2, 3}

print(a - b)
print(b - a)
```

Output

```text
{1}
{3}
```

The order matters.

---

## ❌ Expecting Methods to Modify the Original Set

```python
a = {1, 2}
b = {2, 3}

a.union(b)

print(a)
```

Output

```text
{1, 2}
```

`union()` returns a new set; it does **not** change `a`.

---

# 💡 Best Practices

- Use the method names when readability is important.
- Use operators (`|`, `&`, `-`, `^`) when writing concise code.
- Remember that these methods return **new sets**.

---

# 🚀 Pro Tips

You can combine multiple sets.

```python
a = {1}
b = {2}
c = {3}

print(a.union(b, c))
```

Output

```text
{1, 2, 3}
```
